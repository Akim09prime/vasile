# **App Name**: Carvello Atelier

## Core Features:

- Project Portfolio Timeline: Display a timeline of completed projects, sortable by date and filterable by project type. Each project links to a detailed project page.
- Best-of Furniture Gallery: Showcase a gallery of featured furniture pieces, rated with stars (0-5). Allow filtering by piece type, room, and stage.
- Admin CMS: A comprehensive content management system (CMS) allowing administrators to manage projects, images, categories, pages, menus, footer content, SEO settings, and leads.
- Dynamic Page Builder: Enable creation and management of custom pages with various section types (text, imageText, cards, steps, CTA, galleryStrip, FAQ, testimonials) through the admin CMS.
- Leads Management: Capture and manage quote requests submitted through the 'Cerere ofertÄƒ' form, storing them in the leads collection and allowing status updates within the admin panel.
- Theme Management: Manage light and dark themes and persist the chosen theme and language through local storage.
- Automated Alt Text: Automatically generate descriptive alt text for uploaded images using a generative AI tool. The AI tool considers the image content and assigns alt text for better accessibility and SEO.

## Style Guidelines:

- Primary color: Champagne gold (#B8923B) for a touch of luxury and brand recognition.
- Background color: Warm ivory (#F5F0E8) to provide a welcoming and sophisticated backdrop.
- Accent color: Light gold (#D6C08B), for buttons and to highlight interactive elements.
- Headings font: 'Playfair' (serif) for headlines and titles, lending an elegant and premium feel.
- Body font: 'PT Sans' (sans-serif) for body text to ensure readability and a modern look.
- Use minimalist line icons with a weight that complements the typography for a clean and modern aesthetic.
- Implement a clean, grid-based layout with generous spacing to enhance visual clarity and user experience.
- Subtle fade and upward motion transitions on page load and section reveal, providing a smooth and refined user experience.
